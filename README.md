# 🤖 FormGenerator

A web application that dynamically generates forms from user inputs using the power of Artificial Intelligence.

## 🚀 Live Demo
You can test the application immediately without any installation:
👉 **[form-gen.infinityfree.me](http://form-gen.infinityfree.me)**

## ✨ Features
- **AI-Powered Form Generation (`generate_form.php`):** Automatically creates form fields based on your descriptions and scenarios using AI.
- **User System:** Secure registration and login infrastructure via `signup.php` and `login.php`, allowing users to create and manage their own forms.
- **Form Sharing (`public_form.php`):** Easily export and share your generated forms with your target audience.
- **Data Collection (`save_answers.php`):** Securely receive and store responses from users who fill out your forms.

## 🛠️ Technologies
- **Backend:** PHP
- **Frontend:** HTML / CSS
- **AI:** Gemini API
- **Server & Hosting:** InfinityFree

## 💻 Installation (For Developers)
If you want to run the project locally on your computer, follow these steps:

1. Clone this repository:
   ```bash
   git clone [https://github.com/10Furkan/FormGenerator.git](https://github.com/10Furkan/FormGenerator.git)
   ```
2. Move the project files to the `htdocs` or `www` folder of a local PHP server environment like XAMPP, WAMP, or MAMP.
3. Create your database and import the database.sql file from the main directory.
4. Add your API Key required for the AI integration to the corresponding PHP file.
5. Open your browser and go to `http://localhost/FormGenerator` to start the project.
